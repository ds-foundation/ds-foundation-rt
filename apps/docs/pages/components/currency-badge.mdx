# CurrencyBadge

A compact monospace badge displaying a currency code (USD, EUR, GBP). Pair with `MonoAmount` to show the denomination alongside the value.

## Usage

```tsx
import { MonoAmount, CurrencyBadge } from '@ds-foundation/react';

<span style={{ display: 'inline-flex', alignItems: 'center', gap: 4 }}>
  <MonoAmount value={4500000} currency="EUR" size="lg" />
  <CurrencyBadge currency="EUR" />
</span>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `currency` | `'USD' \| 'EUR' \| 'GBP'` | — | Required. Currency code to display |
| `...rest` | `HTMLAttributes<HTMLSpanElement>` | — | Forwarded to root span |

## Accessibility

- `aria-label` set to `Currency: {code}` for screen readers
