# StateBadge

Displays a current state with an optional next state arrow. Used in settlement workflows to show both where an instruction is now and where it's heading.

## Usage

```tsx
import { StateBadge } from '@ds-foundation/react';

<StateBadge state="Matching" intent="info" />
<StateBadge state="Pending" intent="warning" nextState="Confirmed" />
<StateBadge state="Failed" intent="error" size="sm" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `state` | `string` | — | Required. The current state label |
| `intent` | `'info' \| 'warning' \| 'success' \| 'error' \| 'neutral'` | — | Required. Determines colour |
| `nextState` | `string` | — | Optional. Shows a `→ nextState` annotation |
| `size` | `'sm' \| 'md'` | `'md'` | sm = xs font, md = sm font |
| `...rest` | `HTMLAttributes<HTMLSpanElement>` | — | Forwarded to root span; ref is also forwarded |

## Accessibility

- `role="status"` and `aria-label` built in. Label includes `nextState` if provided
