# StatusPill

Displays the current workflow status of a settlement instruction as a compact pill badge. Maps each `InstructionStatus` to a pre-configured label, colour, and border.

## Usage

```tsx
import { StatusPill } from '@ds-foundation/react';

<StatusPill status="in_payments" />
<StatusPill status="bank_confirmed" />
<StatusPill status="failed" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `status` | `InstructionStatus` | — | Required. One of: `submitted`, `in_payments`, `first_approval`, `second_approval`, `sent_to_bank`, `bank_confirmed`, `failed`, `rejected` |
| `...rest` | `HTMLAttributes<HTMLSpanElement>` | — | Forwarded to root span; ref is also forwarded |

## Accessibility

- `role="status"` on the root span
- `aria-label` is set to `Status: {label}` — overridable via `...rest`
