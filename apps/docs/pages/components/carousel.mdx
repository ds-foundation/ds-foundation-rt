# Carousel

Scrollable slide container for image galleries, feature highlights, and card sliders.

## Usage

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselPrevious,
  CarouselNext,
} from '@ds-foundation/react';

<Carousel className="w-full max-w-sm">
  <CarouselContent>
    {items.map((item, index) => (
      <CarouselItem key={index}>
        <div className="p-6">{item}</div>
      </CarouselItem>
    ))}
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## Props

### Carousel (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Scroll axis |
| `opts` | `EmblaOptionsType` | — | Embla carousel options (loop, align, etc.) |
| `plugins` | `EmblaPluginType[]` | — | Embla plugins |
| `setApi` | `(api: CarouselApi) => void` | — | Receive the embla API for programmatic control |

### CarouselItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Override basis — e.g. `basis-1/3` for 3-up layouts |

### CarouselPrevious / CarouselNext

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `ButtonVariant` | `"outline"` | Button variant |
| `size` | `ButtonSize` | `"icon"` | Button size |

## Accessibility

- Root carries `role="region"` and `aria-roledescription="carousel"` — add an `aria-label` to describe the slides
- Each slide has `role="group"` and `aria-roledescription="slide"`
- Prev/Next buttons include sr-only labels; they are disabled at the scroll boundary
- `ArrowLeft` / `ArrowRight` keys navigate slides when focus is within the carousel
