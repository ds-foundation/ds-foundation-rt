# UrgencyBadge

Displays settlement urgency as a coloured text badge: Critical, Watch, Clear, or Skip-node. Uses the same urgency semantic as `StatusRing` but as a full text label.

## Usage

```tsx
import { UrgencyBadge } from '@ds-foundation/react';

<UrgencyBadge urgency="critical" />
<UrgencyBadge urgency="watch" label="Monitor" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `urgency` | `'critical' \| 'watch' \| 'clear' \| 'skip'` | — | Required. Determines colour |
| `label` | `string` | — | Optional. Overrides the default label text |
| `...rest` | `HTMLAttributes<HTMLSpanElement>` | — | Forwarded to root span |

## Accessibility

- `aria-label` set to `Urgency: {label}`
