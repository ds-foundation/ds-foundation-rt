# NavigationMenu

Horizontal site navigation with animated flyout dropdown panels, built on Radix UI.

## Usage

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
  navigationMenuTriggerStyle,
} from '@ds-foundation/react';

<NavigationMenu>
  <NavigationMenuList>
    <NavigationMenuItem>
      <NavigationMenuTrigger>Products</NavigationMenuTrigger>
      <NavigationMenuContent>
        <ul className="grid gap-3 p-6 md:w-[400px]">
          <li>
            <NavigationMenuLink href="/overview">Overview</NavigationMenuLink>
          </li>
        </ul>
      </NavigationMenuContent>
    </NavigationMenuItem>
    <NavigationMenuItem>
      <NavigationMenuLink className={navigationMenuTriggerStyle()} href="/docs">
        Docs
      </NavigationMenuLink>
    </NavigationMenuItem>
  </NavigationMenuList>
</NavigationMenu>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional classes for the root element |
| `children` | `ReactNode` | — | NavigationMenuList and other children |

### NavigationMenuTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional classes |
| `children` | `ReactNode` | — | Trigger label content |

### NavigationMenuContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional classes |
| `children` | `ReactNode` | — | Dropdown panel content |

### NavigationMenuLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | — | Navigation destination |
| `className` | `string` | — | Additional classes; use `navigationMenuTriggerStyle()` for consistent top-level link styling |
| `asChild` | `boolean` | `false` | Render as child element |

## Accessibility

- The root renders as a `<nav>` element with implicit `navigation` landmark role.
- Triggers manage `aria-expanded` and `aria-haspopup` automatically via Radix.
- Arrow keys navigate between top-level items; Enter/Space open flyouts.
- `NavigationMenuIndicator` provides a visible pointer to the active item.
- Focus is trapped inside the open viewport; Escape closes the panel and returns focus to the trigger.
