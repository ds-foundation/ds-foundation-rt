# Tag

Coloured label chip with optional remove button and leading icon.

## Usage

```tsx
import { Tag } from '@ds-foundation/react';

// Basic
<Tag>Finance</Tag>

// Coloured variant
<Tag variant="blue">Pending</Tag>
<Tag variant="green">Settled</Tag>
<Tag variant="error">Failed</Tag>

// Removable
<Tag variant="purple" onRemove={() => removeTag('label')}>
  Q2 Report
</Tag>

// With icon
import { DollarSign } from 'lucide-react';
<Tag variant="blue" icon={<DollarSign className="w-3 h-3" />}>
  USD
</Tag>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "blue" \| "green" \| "error" \| "orange" \| "purple"` | `"default"` | Colour scheme |
| `onRemove` | `() => void` | — | If provided, renders a close button |
| `icon` | `React.ReactNode` | — | Leading icon slot |
| `children` | `React.ReactNode` | — | Tag label text |
| `className` | `string` | — | Additional classes |

## Accessibility

- The tag renders as a `<span>` — purely presentational
- When `onRemove` is provided, a `<button type="button" aria-label="Remove">` is rendered
- The remove button has a visible focus ring and is keyboard accessible
- Do not hide the remove button with `opacity-0` — it must always be discoverable by keyboard users
