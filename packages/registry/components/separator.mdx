# Separator

Horizontal or vertical divider for visually separating content sections, built on Radix UI.

## Usage

```tsx
import { Separator } from '@ds-foundation/react';

// Horizontal (default)
<Separator className="my-4" />

// Vertical — requires explicit height on parent or self
<div className="flex items-center gap-4 h-5">
  <span>Left</span>
  <Separator orientation="vertical" />
  <span>Right</span>
</div>

// Semantic separator (announced by screen readers)
<Separator decorative={false} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Direction of the divider |
| `decorative` | `boolean` | `true` | When true, hidden from accessibility tree (role="none") |
| `className` | `string` | — | Additional classes for color, margin, or size |

## Accessibility

- When `decorative={true}` (default), the element has `role="none"` and is hidden from screen readers — correct for most visual separators.
- When `decorative={false}`, the element has `role="separator"` and `aria-orientation`, making it discoverable and announceable by assistive technology.
- Horizontal renders as `h-[1px] w-full`; vertical as `h-full w-[1px]`.
