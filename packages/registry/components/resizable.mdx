# Resizable

Drag-to-resize panel layout for split-pane interfaces, built on react-resizable-panels.

## Usage

```tsx
import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from '@ds-foundation/react';

<ResizablePanelGroup direction="horizontal" className="min-h-[200px] rounded-lg border">
  <ResizablePanel defaultSize={50}>
    <div className="flex h-full items-center justify-center p-6">
      <span className="font-semibold">Left panel</span>
    </div>
  </ResizablePanel>
  <ResizableHandle withHandle />
  <ResizablePanel defaultSize={50}>
    <div className="flex h-full items-center justify-center p-6">
      <span className="font-semibold">Right panel</span>
    </div>
  </ResizablePanel>
</ResizablePanelGroup>
```

### Vertical layout

```tsx
<ResizablePanelGroup direction="vertical" className="min-h-[400px]">
  <ResizablePanel defaultSize={30}>Top</ResizablePanel>
  <ResizableHandle />
  <ResizablePanel defaultSize={70}>Bottom</ResizablePanel>
</ResizablePanelGroup>
```

## Props

### ResizablePanelGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `"horizontal" \| "vertical"` | `"horizontal"` | Axis along which panels resize |
| `className` | `string` | — | Additional classes |
| `onLayout` | `(sizes: number[]) => void` | — | Callback when panel sizes change |

### ResizablePanel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultSize` | `number` | — | Initial panel size as a percentage of the group |
| `minSize` | `number` | — | Minimum allowed size (percentage) |
| `maxSize` | `number` | — | Maximum allowed size (percentage) |
| `collapsible` | `boolean` | `false` | Allow the panel to collapse to zero |
| `className` | `string` | — | Additional classes |

### ResizableHandle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `withHandle` | `boolean` | `false` | Show a visible grip icon on the divider |
| `className` | `string` | — | Additional classes |

## Accessibility

- `ResizableHandle` renders with `role="separator"` and appropriate `aria-orientation`.
- The handle is focusable; arrow keys resize the adjacent panels when it has focus.
- A visible focus ring is applied via `focus-visible:ring-1 focus-visible:ring-ring`.
- Use `withHandle` to provide a visible affordance that the divider is draggable.
