# Button

A styled button component with multiple variants for different semantic contexts.

## Usage

```tsx
import { Button } from '@ds-foundation/react';

<Button variant="default">Save</Button>
<Button variant="destructive">Delete</Button>
<Button variant="outline">Cancel</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Learn more</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Visual style |
| `size` | `"sm" \| "default" \| "lg" \| "icon"` | `"default"` | Size preset |
| `asChild` | `boolean` | `false` | Render as child element via Radix Slot |
| `...rest` | `ButtonHTMLAttributes` | — | Forwarded to button element |

## Accessibility

- Native `<button>` element — keyboard and ARIA built in
- Use `aria-label` when button content is icon-only
