# Collapsible

Show/hide a single content section, built on Radix UI CollapsiblePrimitive.

## Usage

```tsx
import {
  Collapsible,
  CollapsibleTrigger,
  CollapsibleContent,
} from '@ds-foundation/react';
import { Button } from '@ds-foundation/react';

<Collapsible>
  <CollapsibleTrigger asChild>
    <Button variant="ghost">Toggle filters</Button>
  </CollapsibleTrigger>
  <CollapsibleContent>
    <div className="space-y-2 pt-4">
      <p>Filter content goes here.</p>
    </div>
  </CollapsibleContent>
</Collapsible>

{/* Controlled */}
<Collapsible open={isOpen} onOpenChange={setIsOpen}>
  <CollapsibleTrigger>Advanced options</CollapsibleTrigger>
  <CollapsibleContent>...</CollapsibleContent>
</Collapsible>
```

## Props

### Collapsible (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Uncontrolled initial open state |
| `onOpenChange` | `(open: boolean) => void` | — | Called when open state changes |
| `disabled` | `boolean` | `false` | Prevents toggling |

### CollapsibleTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element (e.g. existing `<Button>`) |

### CollapsibleContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `...rest` | `ComponentPropsWithoutRef<CollapsiblePrimitive.Content>` | — | Forwarded to Radix content |

## Accessibility

- `aria-expanded` on the trigger reflects open/closed state automatically
- `aria-controls` links the trigger to the content region
- Keyboard: `Space` / `Enter` on the trigger toggles the content
