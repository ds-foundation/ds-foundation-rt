# Badge

Compact pill label for status, category, and count display, built with CVA.

## Usage

```tsx
import { Badge } from '@ds-foundation/react';

<Badge>Active</Badge>
<Badge variant="secondary">Draft</Badge>
<Badge variant="destructive">Blocked</Badge>
<Badge variant="success">Completed</Badge>
<Badge variant="warning">Pending</Badge>
<Badge variant="info">New</Badge>
<Badge variant="outline">Tag</Badge>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "success" \| "warning" \| "info" \| "pink"` | `"default"` | Visual style and semantic color |
| `...rest` | `HTMLAttributes<HTMLDivElement>` | — | Forwarded to the root `<div>` |

## Accessibility

- Badge is non-interactive — it renders as a `<div>`, not a `<button>`
- Always pair a color variant with a text label — do not rely on color alone to convey meaning (WCAG 1.4.1)
- If the badge count updates dynamically (e.g. notification count), wrap in a container with `aria-live="polite"` so changes are announced
- Add `aria-label` when badge content is a number or acronym without surrounding context
