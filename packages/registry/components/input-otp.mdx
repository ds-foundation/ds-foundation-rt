# InputOTP

One-time password entry with individual character slots built on input-otp.

## Usage

```tsx
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSlot,
  InputOTPSeparator,
} from '@ds-foundation/react';
import { REGEXP_ONLY_DIGITS } from 'input-otp';

// 6-digit code
<InputOTP maxLength={6} pattern={REGEXP_ONLY_DIGITS}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
  </InputOTPGroup>
  <InputOTPSeparator />
  <InputOTPGroup>
    <InputOTPSlot index={3} />
    <InputOTPSlot index={4} />
    <InputOTPSlot index={5} />
  </InputOTPGroup>
</InputOTP>

// Controlled
<InputOTP
  maxLength={4}
  value={code}
  onChange={setCode}
>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
    <InputOTPSlot index={3} />
  </InputOTPGroup>
</InputOTP>
```

## Props

### InputOTP

Accepts all props from the `input-otp` `OTPInput` component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxLength` | `number` | — | **Required.** Total number of OTP characters |
| `pattern` | `string` | — | Regex pattern restricting allowed input (e.g. `REGEXP_ONLY_DIGITS`) |
| `value` | `string` | — | Controlled value |
| `onChange` | `(value: string) => void` | — | Called on each keystroke |
| `disabled` | `boolean` | `false` | Disables input; container dims via `has-[:disabled]:opacity-50` |

### InputOTPSlot

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | — | **Required.** Position of this slot in the OTP sequence |

## Accessibility

- The underlying element is a single native `<input>` — screen readers interact with it as a single field.
- The slot UI is purely visual decoration.
- `InputOTPSeparator` renders a `<div role="separator">` with a `Dot` icon.
- The active (focused) slot displays a `ring-2` highlight and an animated caret.
- Disabled state applies `opacity-50` to the entire container via `has-[:disabled]`.
