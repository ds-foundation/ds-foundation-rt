# Label

Accessible form label built on the Radix UI Label primitive.

## Usage

```tsx
import { Label } from '@ds-foundation/react';
import { Input } from '@ds-foundation/react';

// Basic usage
<div className="space-y-1">
  <Label htmlFor="username">Username</Label>
  <Input id="username" type="text" placeholder="johndoe" />
</div>

// Auto-dims when input is disabled (peer pattern)
<div className="space-y-1">
  <Label htmlFor="locked">Locked field</Label>
  <Input id="locked" disabled />
</div>
```

## Props

`Label` accepts all Radix `Label.Root` props plus standard HTML label attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | — | Id of the associated form control |
| `className` | `string` | — | Additional Tailwind classes |

## Accessibility

- Renders a native `<label>` element — clicking it focuses the associated input.
- `htmlFor` must be set to the `id` of the associated control.
- `peer-disabled:opacity-70` visually dims the label when the adjacent input is disabled.
- Inside `Form`, use `FormLabel` which sets `htmlFor` automatically via form context.
