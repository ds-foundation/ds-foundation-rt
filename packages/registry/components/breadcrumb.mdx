# Breadcrumb

Navigation breadcrumb trail showing the user's current location in the site hierarchy.

## Usage

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator,
  BreadcrumbEllipsis,
} from '@ds-foundation/react';

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="/payments">Payments</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Transaction Detail</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Props

### Breadcrumb (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `separator` | `ReactNode` | — | Custom separator element |
| `...rest` | `ComponentPropsWithoutRef<"nav">` | — | Forwarded to `<nav>` |

### BreadcrumbLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element via Radix Slot (e.g. a router `<Link>`) |
| `href` | `string` | — | Navigation target |
| `...rest` | `ComponentPropsWithoutRef<"a">` | — | Forwarded to `<a>` |

### BreadcrumbPage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `...rest` | `ComponentPropsWithoutRef<"span">` | — | Forwarded to `<span>` with `aria-current="page"` |

### BreadcrumbSeparator / BreadcrumbEllipsis / BreadcrumbList / BreadcrumbItem

Accept all standard HTML attributes plus `className`.

## Accessibility

- Root `<nav>` carries `aria-label="breadcrumb"` automatically
- `BreadcrumbPage` receives `aria-current="page"` and `aria-disabled="true"` — screen readers identify it as the current location
- Separators and the ellipsis are hidden from assistive technology via `role="presentation"` and `aria-hidden="true"`
- `BreadcrumbEllipsis` includes a visually hidden `<span className="sr-only">More</span>` for screen readers
