# CurrencyBadge

A compact pill badge that displays a currency code (USD, EUR, GBP) in monospace font. Purely informational — no interaction behavior.

## Usage

Pair with `MonoAmount` to label the denomination of a financial value.

```tsx
import { CurrencyBadge } from '@ds-foundation/react';

<CurrencyBadge currency="USD" />
<CurrencyBadge currency="EUR" />
<CurrencyBadge currency="GBP" />
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `currency` | `'USD' \| 'EUR' \| 'GBP'` | The currency code to display |

## Accessibility

Includes `aria-label="Currency: {currency}"` so screen readers can announce the denomination without relying on visual-only context.
