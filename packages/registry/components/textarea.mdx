# Textarea

Multi-line text input styled to match the design system's form controls.

## Usage

```tsx
import { Textarea } from '@ds-foundation/react';
import { Label } from '@ds-foundation/react';

// Basic
<div className="grid gap-1.5">
  <Label htmlFor="notes">Notes</Label>
  <Textarea id="notes" placeholder="Add a note…" />
</div>

// Controlled
const [value, setValue] = React.useState('');

<Textarea
  value={value}
  onChange={(e) => setValue(e.target.value)}
  rows={6}
  placeholder="Describe the issue…"
/>

// Disabled
<Textarea disabled placeholder="Read only" />
```

## Props

All native HTML `<textarea>` attributes are accepted, including:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `rows` | `number` | — | Number of visible text rows |
| `placeholder` | `string` | — | Placeholder text |
| `disabled` | `boolean` | `false` | Disables input and changes cursor |
| `value` | `string` | — | Controlled value |
| `onChange` | `React.ChangeEventHandler<HTMLTextAreaElement>` | — | Change handler |
| `maxLength` | `number` | — | Maximum character count |
| `className` | `string` | — | Additional classes |

## Accessibility

- Always associate a `<Label>` using `htmlFor` matching the textarea's `id`
- Add `aria-invalid="true"` and `aria-describedby` pointing to an error message element when validation fails
- The textarea has a minimum height of 80px and is full-width by default
