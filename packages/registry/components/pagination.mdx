# Pagination

Page-based navigation controls for moving through paged data sets.

## Usage

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from '@ds-foundation/react';

<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="/page/1" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="/page/1">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="/page/2" isActive>2</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="/page/3">3</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationEllipsis />
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="/page/3" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```

## Props

### PaginationLink

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isActive` | `boolean` | `false` | Marks the current page; applies `aria-current="page"` and outline variant |
| `size` | `ButtonProps["size"]` | `"icon"` | Button size variant |
| `href` | `string` | — | Page destination URL |
| `className` | `string` | — | Additional classes |

### PaginationPrevious / PaginationNext

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | — | Destination URL |
| `className` | `string` | — | Additional classes |

## Accessibility

- Root `<nav>` has `role="navigation"` and `aria-label="pagination"` automatically.
- Active page link receives `aria-current="page"`.
- Previous and Next buttons have descriptive `aria-label` values.
- `PaginationEllipsis` is `aria-hidden`; a visually hidden `<span>` reads "More pages" for screen readers.
