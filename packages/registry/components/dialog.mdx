# Dialog

Modal dialog built on Radix UI primitives with overlay, animations, and built-in close button.

## Usage

```tsx
import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogFooter,
  DialogTitle,
  DialogDescription,
  DialogClose,
} from '@ds-foundation/react';
import { Button } from '@ds-foundation/react';

<Dialog>
  <DialogTrigger asChild>
    <Button variant="outline">Open Dialog</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Are you sure?</DialogTitle>
      <DialogDescription>
        This action cannot be undone.
      </DialogDescription>
    </DialogHeader>
    <DialogFooter>
      <DialogClose asChild>
        <Button variant="outline">Cancel</Button>
      </DialogClose>
      <Button>Confirm</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

## Props

### DialogContent

Accepts all Radix `Dialog.Content` props plus `className`.

### DialogHeader / DialogFooter

Plain `<div>` wrappers; accept all `HTMLDivElement` props.

| Component | Layout |
|-----------|--------|
| `DialogHeader` | `flex flex-col space-y-1.5 text-center sm:text-left` |
| `DialogFooter` | `flex flex-col-reverse sm:flex-row sm:justify-end sm:space-x-2` |

### DialogTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional Tailwind classes |

### DialogDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional Tailwind classes |

## Accessibility

- `DialogContent` has `role="dialog"`, `aria-modal="true"`, `aria-labelledby` (DialogTitle), and `aria-describedby` (DialogDescription).
- Focus is trapped inside the dialog while open; restored to trigger on close.
- Escape key closes the dialog.
- The built-in close button (X icon) includes an `sr-only` label "Close".
