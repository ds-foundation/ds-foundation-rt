# Command

Searchable command palette for keyboard-driven navigation and action execution, built on cmdk.

## Usage

```tsx
import {
  Command,
  CommandDialog,
  CommandInput,
  CommandList,
  CommandEmpty,
  CommandGroup,
  CommandItem,
  CommandShortcut,
  CommandSeparator,
} from '@ds-foundation/react';

{/* Inline */}
<Command>
  <CommandInput placeholder="Search commands..." />
  <CommandList>
    <CommandEmpty>No results found.</CommandEmpty>
    <CommandGroup heading="Suggestions">
      <CommandItem>
        Calendar
        <CommandShortcut>⌘C</CommandShortcut>
      </CommandItem>
      <CommandItem>Search Payments</CommandItem>
    </CommandGroup>
    <CommandSeparator />
    <CommandGroup heading="Settings">
      <CommandItem>Profile</CommandItem>
      <CommandItem>Billing</CommandItem>
    </CommandGroup>
  </CommandList>
</Command>

{/* Modal palette (Cmd+K) */}
<CommandDialog open={open} onOpenChange={setOpen}>
  <CommandInput placeholder="Type a command..." />
  <CommandList>
    <CommandEmpty>No results.</CommandEmpty>
    <CommandGroup heading="Actions">
      <CommandItem onSelect={() => runAction()}>New Payment</CommandItem>
    </CommandGroup>
  </CommandList>
</CommandDialog>
```

## Props

### Command (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `filter` | `(value: string, search: string) => number` | built-in | Custom filter function |
| `shouldFilter` | `boolean` | `true` | Enable/disable built-in filtering |
| `value` | `string` | — | Controlled selected item value |
| `onValueChange` | `(value: string) => void` | — | Called when selected item changes |
| `loop` | `boolean` | `false` | Wrap arrow key navigation at ends |

### CommandDialog

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Dialog open state |
| `onOpenChange` | `(open: boolean) => void` | — | Called when dialog open state changes |
| `...rest` | `DialogProps` | — | All Radix Dialog props |

### CommandItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Item value used for filtering |
| `onSelect` | `(value: string) => void` | — | Called when item is activated |
| `disabled` | `boolean` | `false` | Prevents selection |

### CommandGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `heading` | `string` | — | Section label text |

## Accessibility

- `CommandInput` has `role="combobox"` with `aria-expanded` and `aria-controls` pointing to the list
- `CommandList` has `role="listbox"`; `CommandItem` has `role="option"` with `aria-selected`
- Keyboard: `Arrow Up/Down` navigate items; `Enter` activates; `Escape` closes dialog
- Items are filtered in real time as the user types in `CommandInput`
