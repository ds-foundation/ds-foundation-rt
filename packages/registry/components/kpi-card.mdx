# KpiCard

Metric display card for dashboard KPIs with optional trend indicator.

## Usage

```tsx
import { KpiCard } from '@ds-foundation/react';
import { DollarSignIcon } from 'lucide-react';

// Minimal
<KpiCard label="Total Revenue" value="$1,240,000" />

// With trend and icon
<KpiCard
  label="Active Users"
  value={4821}
  trend={{ direction: "up", label: "+12% this week" }}
  icon={<DollarSignIcon className="h-5 w-5" />}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | **Required.** Metric name shown above the value |
| `value` | `string \| number` | — | **Required.** Metric value; numbers are formatted with `toLocaleString()` |
| `trend` | `{ direction: 'up' \| 'down' \| 'neutral'; label: string }` | — | Optional trend badge |
| `icon` | `React.ReactNode` | — | Decorative icon shown top-right |
| `className` | `string` | — | Additional classes on the root container |

## Accessibility

- The component is presentational — no interactive ARIA roles.
- The icon is decorative and carries no ARIA label.
- Trend direction is conveyed via both icon and visible text label.
