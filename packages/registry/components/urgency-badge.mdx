# UrgencyBadge

A color-coded pill badge that labels the urgency level of a settlement instruction. Pairs with `StatusRing` for a complete visual + text urgency indicator.

## Usage

```tsx
import { UrgencyBadge } from '@ds-foundation/react';

<UrgencyBadge urgency="critical" />
<UrgencyBadge urgency="watch" />
<UrgencyBadge urgency="clear" />
<UrgencyBadge urgency="skip" />

// Custom label
<UrgencyBadge urgency="critical" label="Urgent" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `urgency` | `'critical' \| 'watch' \| 'clear' \| 'skip'` | — | Urgency level |
| `label` | `string` | — | Overrides the default label text |

## Default labels

| Urgency | Default label |
|---------|--------------|
| `critical` | Critical |
| `watch` | Watch |
| `clear` | Clear |
| `skip` | Skip-node |

## Accessibility

Includes `aria-label="Urgency: {label}"`. Urgency is conveyed through both color and visible text — color alone is never the sole indicator.
