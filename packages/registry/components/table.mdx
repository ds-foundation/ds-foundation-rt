# Table

Composable semantic data table with scrollable container, header, body, footer, and caption.

## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableHead,
  TableRow,
  TableCell,
  TableCaption,
} from '@ds-foundation/react';

<Table>
  <TableCaption>Recent transactions</TableCaption>
  <TableHeader>
    <TableRow>
      <TableHead>Date</TableHead>
      <TableHead>Description</TableHead>
      <TableHead className="text-right">Amount</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>2026-04-01</TableCell>
      <TableCell>Wire transfer</TableCell>
      <TableCell className="text-right">$12,500.00</TableCell>
    </TableRow>
  </TableBody>
  <TableFooter>
    <TableRow>
      <TableCell colSpan={2}>Total</TableCell>
      <TableCell className="text-right">$12,500.00</TableCell>
    </TableRow>
  </TableFooter>
</Table>
```

## Props

All parts accept `className` and standard HTML element props for their respective elements.

### Table

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Applied to the inner `<table>` element |

### TableHead

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Applied to the `<th>` element |

### TableCell

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `colSpan` | `number` | — | Spans multiple columns |
| `className` | `string` | — | Applied to the `<td>` element |

## Accessibility

- Native HTML table semantics provide implicit ARIA roles — no extra attributes required
- `<TableHead>` renders `<th>` elements; browsers expose these as column headers to screen readers
- `<TableCaption>` renders a `<caption>` element, giving the table an accessible name
- Rows with `data-[state=selected]` apply a muted background for visual selection state
