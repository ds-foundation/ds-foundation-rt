# BankingWindowDot

A small dot indicator showing whether a banking window is open, closing, or closed. The `closing` state pulses to draw attention to the upcoming cutoff.

## Usage

```tsx
import { BankingWindowDot } from '@ds-foundation/react';

<BankingWindowDot status="open" />
<BankingWindowDot status="closing" />
<BankingWindowDot status="closed" />

// Larger size
<BankingWindowDot status="open" size={10} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `status` | `'open' \| 'closing' \| 'closed'` | — | Banking window state |
| `size` | `number` | `6` | Dot diameter in pixels |

## States

| Status | Color | Animation |
|--------|-------|-----------|
| `open` | Success green | None |
| `closing` | Warning amber | Pulse ring |
| `closed` | Tertiary gray | None |

## Accessibility

The dot is `aria-hidden="true"`. Always pair with visible text (e.g., "NY Open", "London Closing") so the window state is not communicated by color alone.
