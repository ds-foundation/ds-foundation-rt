# Drawer

Slide-in panel (defaults to bottom) built on the vaul library.

## Usage

```tsx
import {
  Drawer,
  DrawerTrigger,
  DrawerContent,
  DrawerHeader,
  DrawerFooter,
  DrawerTitle,
  DrawerDescription,
  DrawerClose,
} from '@ds-foundation/react';
import { Button } from '@ds-foundation/react';

<Drawer>
  <DrawerTrigger asChild>
    <Button variant="outline">Open Drawer</Button>
  </DrawerTrigger>
  <DrawerContent>
    <DrawerHeader>
      <DrawerTitle>Edit Profile</DrawerTitle>
      <DrawerDescription>
        Make changes to your profile here.
      </DrawerDescription>
    </DrawerHeader>
    <div className="p-4">
      {/* form content */}
    </div>
    <DrawerFooter>
      <Button>Save</Button>
      <DrawerClose asChild>
        <Button variant="outline">Cancel</Button>
      </DrawerClose>
    </DrawerFooter>
  </DrawerContent>
</Drawer>
```

## Props

### Drawer (root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `shouldScaleBackground` | `boolean` | `true` | Scales the background page when drawer opens |
| `direction` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | Edge the drawer slides in from |

### DrawerContent

Accepts all vaul `Drawer.Content` props plus `className`. A drag-handle bar is rendered automatically at the top.

### DrawerHeader / DrawerFooter

Plain `<div>` wrappers accepting all `HTMLDivElement` props.

| Component | Layout |
|-----------|--------|
| `DrawerHeader` | `grid gap-1.5 p-4 text-center sm:text-left` |
| `DrawerFooter` | `mt-auto flex flex-col gap-2 p-4` |

## Accessibility

- `DrawerContent` renders with `role="dialog"`, `aria-labelledby`, and `aria-describedby`.
- Focus is trapped inside the drawer; Escape or the close button dismisses it.
- The drag-handle is a decorative `<div>` with no ARIA role.
