# Avatar

Circular user image with automatic fallback, built on Radix UI AvatarPrimitive.

## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback } from '@ds-foundation/react';

<Avatar>
  <AvatarImage src="https://example.com/user.jpg" alt="Jane Smith" />
  <AvatarFallback>JS</AvatarFallback>
</Avatar>

{/* Icon fallback */}
<Avatar>
  <AvatarImage src={user.avatarUrl} alt={user.name} />
  <AvatarFallback>
    <UserIcon className="h-5 w-5" />
  </AvatarFallback>
</Avatar>
```

## Props

### Avatar (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Override size or shape (default is `h-10 w-10 rounded-full`) |
| `...rest` | `ComponentPropsWithoutRef<AvatarPrimitive.Root>` | — | Forwarded to Radix root |

### AvatarImage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Image URL |
| `alt` | `string` | — | Descriptive alt text (required for accessibility) |
| `...rest` | `ComponentPropsWithoutRef<AvatarPrimitive.Image>` | — | Forwarded to Radix image |

### AvatarFallback

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delayMs` | `number` | — | Milliseconds to wait before showing fallback (avoids flash) |
| `...rest` | `ComponentPropsWithoutRef<AvatarPrimitive.Fallback>` | — | Forwarded to Radix fallback |

## Accessibility

- Provide meaningful `alt` text on `AvatarImage` — screen readers read this as the image label
- When the avatar appears without an adjacent visible name, add `aria-label` to the root `Avatar`
- Fallback text (initials) must meet 4.5:1 contrast ratio against `bg-muted`
