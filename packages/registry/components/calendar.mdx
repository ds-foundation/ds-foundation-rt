# Calendar

Interactive monthly date picker grid built on react-day-picker v9.

## Usage

```tsx
import * as React from 'react';
import { Calendar } from '@ds-foundation/react';

function DatePicker() {
  const [date, setDate] = React.useState<Date | undefined>();

  return (
    <Calendar
      mode="single"
      selected={date}
      onSelect={setDate}
    />
  );
}

// Date range
function DateRangePicker() {
  const [range, setRange] = React.useState<{ from?: Date; to?: Date }>();

  return (
    <Calendar
      mode="range"
      selected={range}
      onSelect={setRange}
    />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `"single" \| "multiple" \| "range"` | `"single"` | Selection mode |
| `selected` | `Date \| Date[] \| DateRange \| undefined` | — | Currently selected date(s) |
| `onSelect` | `(date: ...) => void` | — | Callback when selection changes |
| `disabled` | `Matcher \| Matcher[]` | — | Dates to disable (function or date/range) |
| `showOutsideDays` | `boolean` | `true` | Show days from adjacent months |
| `captionLayout` | `"label" \| "dropdown" \| "dropdown-years"` | `"label"` | Month/year navigation style |
| `buttonVariant` | `ButtonVariant` | `"ghost"` | Style of the prev/next navigation buttons |
| `...rest` | `ComponentProps<typeof DayPicker>` | — | All react-day-picker props |

## Exports

- `Calendar` — the main component
- `CalendarDayButton` — individual day button, exported for custom composition

## Accessibility

- Month grid uses `role="grid"` with appropriate row/cell roles
- Selected cells carry `aria-selected="true"`; disabled cells carry `aria-disabled="true"`
- Month navigation buttons have accessible labels
- Month/year caption change is announced via `aria-live="polite"`
- Keyboard: `Arrow` keys navigate days; `Enter`/`Space` selects; `Page Up/Down` changes month
