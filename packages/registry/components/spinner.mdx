# Spinner

Animated SVG loading indicator with accessible screen reader announcement.

## Usage

```tsx
import { Spinner } from '@ds-foundation/react';

// Default (md)
<Spinner />

// Small inside a button
<button disabled>
  <Spinner size="sm" />
  Saving…
</button>

// Large overlay
<div className="flex items-center justify-center h-64">
  <Spinner size="lg" />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Controls width/height (16px / 24px / 32px) |
| `className` | `string` | — | Additional classes applied to the SVG element |

## Accessibility

- Outer `<span>` carries `role="status"` and `aria-label="Loading"` so screen readers announce loading state
- A visually hidden `<span className="sr-only">Loading…</span>` provides text for readers that ignore `aria-label` on non-interactive elements
- The SVG itself is purely decorative and has no accessible name
