# Timeline

Vertical list of timestamped events with status-coloured markers and connector lines.

## Usage

```tsx
import { Timeline } from '@ds-foundation/react';
import { CheckCircle } from 'lucide-react';

<Timeline
  items={[
    {
      id: '1',
      title: 'Payment initiated',
      timestamp: '09:00',
      status: 'complete',
    },
    {
      id: '2',
      title: 'Compliance review',
      description: 'Awaiting AML check',
      timestamp: '09:15',
      status: 'active',
    },
    {
      id: '3',
      title: 'Settlement',
      status: 'default',
    },
  ]}
/>

// With custom icon
<Timeline
  items={[
    {
      id: '1',
      icon: <CheckCircle className="w-3 h-3" />,
      title: 'Approved',
      status: 'complete',
    },
  ]}
/>
```

## Props

### Timeline

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `TimelineItem[]` | ŌĆö | Array of event objects (required) |
| `className` | `string` | ŌĆö | Additional classes on the container |

### TimelineItem object

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `id` | `string` | ŌĆö | Unique key (required) |
| `title` | `string` | ŌĆö | Primary event label (required) |
| `description` | `string` | ŌĆö | Secondary description text |
| `timestamp` | `string` | ŌĆö | Date/time string displayed at the right |
| `icon` | `React.ReactNode` | ŌĆö | Icon rendered inside the marker circle |
| `status` | `"default" \| "active" \| "complete" \| "error"` | `"default"` | Controls marker colour |

## Accessibility

- The outer element is an `<ol>` with `role="list"`; each event is a `<li>` with `role="listitem"`
- Status is communicated via colour (grey/blue/green/orange) ŌĆö provide descriptive `title` and `description` text so the meaning is not colour-only
- Connector lines are decorative `<div>` elements excluded from the accessibility tree
