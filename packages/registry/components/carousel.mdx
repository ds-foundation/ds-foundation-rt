# Carousel

Horizontally or vertically scrollable slide container built on embla-carousel-react.

## Usage

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselPrevious,
  CarouselNext,
} from '@ds-foundation/react';
import { Card, CardContent } from '@ds-foundation/react';

<Carousel className="w-full max-w-sm">
  <CarouselContent>
    {items.map((item, index) => (
      <CarouselItem key={index}>
        <Card>
          <CardContent className="flex aspect-square items-center justify-center p-6">
            <span className="text-4xl font-semibold">{item}</span>
          </CardContent>
        </Card>
      </CarouselItem>
    ))}
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## Props

### Carousel (Root)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Scroll axis |
| `opts` | `EmblaOptionsType` | — | Embla carousel options (loop, align, etc.) |
| `plugins` | `EmblaPluginType[]` | — | Embla plugins |
| `setApi` | `(api: CarouselApi) => void` | — | Callback to receive the embla API instance |

### CarouselItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Override basis for partial-slide layouts (e.g. `basis-1/3`) |

### CarouselPrevious / CarouselNext

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `ButtonVariant` | `"outline"` | Button variant |
| `size` | `ButtonSize` | `"icon"` | Button size |

## Exports

- `Carousel`, `CarouselContent`, `CarouselItem`, `CarouselPrevious`, `CarouselNext`
- `type CarouselApi` — the embla carousel API type

## Accessibility

- Root has `role="region"` and `aria-roledescription="carousel"` — add `aria-label` to describe the content
- Each `CarouselItem` has `role="group"` and `aria-roledescription="slide"`
- Previous/Next buttons include visually-hidden labels ("Previous slide" / "Next slide")
- Arrow keys navigate slides; buttons are disabled at the boundary
