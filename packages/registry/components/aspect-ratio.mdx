# AspectRatio

Maintains a fixed width-to-height ratio for media and embedded content, built on Radix UI AspectRatioPrimitive.

## Usage

```tsx
import { AspectRatio } from '@ds-foundation/react';

<div className="w-full max-w-sm">
  <AspectRatio ratio={16 / 9}>
    <img
      src="/hero.jpg"
      alt="Hero image"
      className="h-full w-full object-cover rounded-md"
    />
  </AspectRatio>
</div>

{/* Square avatar container */}
<div className="w-24">
  <AspectRatio ratio={1}>
    <img src="/avatar.png" alt="User avatar" className="h-full w-full object-cover" />
  </AspectRatio>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `ratio` | `number` | `1` | Width divided by height (e.g. `16/9`, `4/3`, `1`) |
| `...rest` | `HTMLAttributes<HTMLDivElement>` | — | Forwarded to the root element |

## Accessibility

- The container is presentational — no interactive ARIA attributes are needed on it
- Child `<img>` elements must carry a descriptive `alt` attribute
- Child videos and iframes should provide accessible labels via `aria-label` or `title`
