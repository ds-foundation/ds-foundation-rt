# StatusRing

A circular dot with a translucent glow ring that encodes urgency at a glance — critical, watch, clear, or skip. Intended as a visual accent, not a standalone indicator.

## Usage

```tsx
import { StatusRing } from '@ds-foundation/react';

<StatusRing urgency="critical" />
<StatusRing urgency="watch" pulse />
<StatusRing urgency="clear" size="sm" />
<StatusRing urgency="skip" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `urgency` | `'critical' \| 'watch' \| 'clear' \| 'skip'` | — | Visual urgency level |
| `size` | `'sm' \| 'md'` | `'md'` | Dot size: sm=8px, md=10px |
| `pulse` | `boolean` | `false` | Enable pulsing animation for active/urgent states |

## Accessibility

Always pair `StatusRing` with a visible text label such as `UrgencyBadge`. The component is `aria-hidden="true"` by design — urgency must not be communicated through color alone.
