# Stepper

Horizontal multi-step progress indicator for wizards and multi-stage forms.

## Usage

```tsx
import { Stepper } from '@ds-foundation/react';

<Stepper
  steps={[
    { label: 'Details', status: 'complete' },
    { label: 'Review', description: 'Check your entries', status: 'active' },
    { label: 'Confirm', status: 'pending' },
  ]}
/>
```

## Props

### Stepper

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `steps` | `Step[]` | — | Array of step objects (required) |
| `className` | `string` | — | Additional classes on the container |

### Step object

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `label` | `string` | — | Step label displayed below the circle |
| `description` | `string` | — | Optional secondary label |
| `status` | `"pending" \| "active" \| "complete" \| "error"` | — | Visual and semantic state of the step |

## Accessibility

- The outer `<ol>` has `role="list"` and each step `<li>` has `role="listitem"`
- The currently active step has `aria-current="step"`
- Connector lines are decorative and hidden from assistive technology
- Status colours (blue for active/complete, orange for error, grey for pending) are supplemented by icon and text cues
