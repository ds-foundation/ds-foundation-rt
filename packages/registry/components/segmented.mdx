# Segmented

Segmented control for selecting one option from a small mutually-exclusive set.

## Usage

```tsx
import { Segmented } from '@ds-foundation/react';

const options = [
  { label: 'List', value: 'list' },
  { label: 'Grid', value: 'grid' },
  { label: 'Map', value: 'map', disabled: true },
];

// Uncontrolled
<Segmented options={options} defaultValue="list" />

// Controlled
const [view, setView] = React.useState('list');

<Segmented
  options={options}
  value={view}
  onChange={setView}
/>
```

### With node labels

```tsx
<Segmented
  options={[
    { label: <><BoldIcon className="h-4 w-4" /> Bold</>, value: 'bold' },
    { label: <><ItalicIcon className="h-4 w-4" /> Italic</>, value: 'italic' },
  ]}
  defaultValue="bold"
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `options` | `SegmentedOption[]` | — | Array of segment definitions (required) |
| `value` | `string` | — | Controlled active value |
| `defaultValue` | `string` | — | Initial active value (uncontrolled) |
| `onChange` | `(value: string) => void` | — | Callback when selection changes |
| `className` | `string` | — | Additional classes for the container |

### SegmentedOption

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | — | Display content for the segment |
| `value` | `string` | — | Unique value identifier |
| `disabled` | `boolean` | `false` | Prevents selection and skips keyboard navigation |

## Accessibility

- Container has `role="radiogroup"`.
- Each segment button has `role="radio"` and `aria-checked` reflecting its active state.
- Roving tabIndex keeps only the active segment in the tab sequence.
- Arrow Left/Right, Home, and End navigate between enabled segments.
- Disabled segments have the native `disabled` attribute and are skipped by arrow key navigation.
